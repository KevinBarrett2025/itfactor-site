Project overview

What this proposal covers

The work centers on the main bathroom and the separate toilet room. The current plan keeps the existing bathtub, improves the room layout, updates finishes, and refreshes the vanity, tile, trim, lighting, and electrical details so the space reads cleaner and works better day to day.

Primary construction direction

Remove the lowered soffit ceiling, close the vanity-side door opening, repair or replace the broken pocket door, and update finishes without replacing the existing tub.

Likely finish direction

Replace floor tile and shower surround tile, extend to a longer double-sink vanity, refresh trim and baseboards, and reuse existing accessories where practical.

Included scope of work

Planned renovation scope

  1. Main bathroom and separate toilet room are included in this proposal.
  2. Remove the lowered soffit ceiling in the main bathroom.
  3. Remove the vanity-side door opening and infill that wall.
  4. Repair or replace the broken pocket door into the toilet room.
  5. Refinish the existing bathtub instead of replacing it.
  6. Replace floor tile throughout the bathroom work area.
  7. Replace shower and tub surround tile.
  8. Finish selected walls with drywall and paint if the final design calls for it.
  9. Likely install a longer double-sink vanity.
  10. Move one outlet, add one outlet, and install two horizontal outlets in the vanity area.
  11. Reuse the existing chandelier as a flush-mounted ceiling fixture.
  12. Replace baseboards and trim.
  13. Salvage and reinstall existing towel bars if they can be reused cleanly.
  14. Add a hand towel ring and minor hardware as needed to finish the room.
